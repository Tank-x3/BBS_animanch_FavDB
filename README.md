# あにまん掲示板_お気に入りスレッドDBツール
あにまん掲示板のお気に入りスレッド情報を抽出してデータベース化します。  
このツールはベータ版です。大幅な変更が行われる可能性があります。  
## 抽出ツール
まず`個人設定`の画面でページを保存してhtml/mhtファイルを保存してください。  
そのファイルをこのツールに読み込ませることで、データベースファイル（`fav_database.json`）を作成します。  
既存のファイルに新規分を追加したり、複数のhtml/mhtファイルから同時にデータを追加することも可能です。
## 編集ツール
抽出ツールで作成したデータベースファイルを読み込むことで、そこに記録されているスレッドが一覧として表示されます。  
スレッドごとにタグ付けしたり、説明文を追加したりして、リンク集のようにすることができます。  
タグ付け等の内容を追記したファイルを保存することができます。  
※今後、DBから「共有用のHTMLファイル（読み取り専用のリンク集ページ）」のようなものを出力できる機能を追加する予定です。
